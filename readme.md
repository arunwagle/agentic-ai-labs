# Agentic AI Lab 🧠⚙️

A personal lab of hands-on experiments with AI agents.

This repository contains demos, prototypes, and explorations using tools like Claude, Databricks Genie Code, and other LLM-based frameworks to understand how agentic systems behave in real-world scenarios.

---

## 🚀 What This Repo Is

- A collection of **small, practical agent demos**
- A space to test **new tools, prompts, and workflows**
- Experiments with **multi-step reasoning and automation**

---

## 🧪 Focus Areas

- Agent orchestration patterns
- Prompt + tool interaction
- Data-driven agents (Databricks, APIs)
